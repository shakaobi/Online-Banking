The application requires Python3 and two 3rd-party modules (passlib and flask) to run.
To install the modules use the commands:
	python -m pip install passlib
	python -m pip install flask

To run the app:
	Navigate to this folder in a terminal.
	Use the command "set FLASK_APP=online_bank"
	Use the command "python -m flask run"
	Navigate to localhost:5000 or http://127.0.0.1:5000/ in a web browser.
